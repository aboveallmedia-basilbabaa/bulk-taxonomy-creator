# Bulk Taxonomy Creator Plugin

## Description

The Bulk Taxonomy Creator is a WordPress plugin that allows you to create post categories and tags in bulk directly from the WordPress admin dashboard. This plugin simplifies the process of adding multiple categories and tags to your WordPress site without the need to associate them with posts immediately.

## Installation

1. Download the plugin ZIP file from the [GitHub repository](https://github.com/aboveallmedia-basilbabaa/bulk-taxonomy-creator.git).
2. Upload the ZIP file through the WordPress admin dashboard.

## Usage

1. After installation, navigate to the WordPress admin dashboard.
2. Go to the "Settings" menu and click on "Bulk Taxonomy Creator."
3. In the "Bulk Taxonomy Creator" settings page, you will find two large input fields:
   - "Categories (comma-separated)": Enter the category names you want to create, separated by commas.
   - "Tags (comma-separated)": Enter the tag names you want to create, separated by commas.
4. Click the "Create Categories and Tags" button to create the specified categories and tags.

## Features

- Bulk creation of categories and tags.
- Simple and user-friendly interface.
- Large input fields for easier data entry.
- Categories and tags are created without immediate association with posts.
